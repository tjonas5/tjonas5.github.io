1 The Binomial and Negative Binomial Distributions

1.1 The Binomial Distribution

In many introductory courses, we make the simplifying assumption of independence in Bernoulli trials. For example, when playing best two-out-of-three legs in a darts match, we would assume Player A has an 70% chance of winning each leg. We write: \[X \sim \text{Binomial}(n=3, p=0.70)\] where

  • \(X\) is the number of legs won
  • \(n\) is the number of trials
  • \(p\) is the probability of the event of interest

Of note, it is not necessarily the case that the players play all three legs – if either player wins legs one and two the match ends.

There are eight possible outcomes for Player A:

  • (win, win, “win”) \(\longrightarrow\) Player A wins
  • (win, win, “lose”) \(\longrightarrow\) Player A wins
  • (win, lose, win) \(\longrightarrow\) Player A wins
  • (win, lose, lose) \(\longrightarrow\) Player A loses
  • (lose, win, win) \(\longrightarrow\) Player A wins
  • (lose, win, lose) \(\longrightarrow\) Player A loses
  • (lose, lose, “win”) \(\longrightarrow\) Player A loses
  • (lose, lose, “lose”) \(\longrightarrow\) Player A loses

Note: I use quotations above to denote outcomes where the last match isn’t actually played.

Use the Binomial distribution to compute the probability that Player A wins the match. Make sure write out your probability statement and work to write it in terms of the distribution function (PMF/CDF) to plot the probability using ggplot.

Solution My probability statement: P(X \(\geq\) 2). Since this is a discrete distribution, I choose to write it using the PMF, which represents it as f(2) + f(3), since the maximum value of x is 3.

1.2 The Negative Binomial Distribution

In intermediate courses, we learn the Negative Binomial distribution that we use to model the number of “failures” until the \(r\)th success. Note that this approach still makes the simplifying assumption of independence in Bernoulli trials.

Specify the two cases necessary to compute the probability. Fully explain these probability statements in the context of a best two-out-of-three match and the Negative Binomial distribution. Finally, compute the probability that Player A wins the match and show you get the same result as the Binomial approach. Make sure write out your probability statement and work to write it in terms of the distribution function (PMF/CDF) to plot the probability using ggplot.

Solution

My probability statement is P(x \(\leq\) 1), which can be rewritten as f(0) + f(1) The two cases necessary to compute are the chances that 2 wins are reached with one loss, and two wins are reached with 0 losses. Summing these together gives the total probability that 2 wins are reached with 1 or 0 losses, which in the context of this best two-out-of-three match means that player A has won the game. The computed probability is the same as the computed binomial distribution one, verifying that this is correct.

2 The Beta Binomial Distribution

One approach to generalize this modeling is to use a Beta-Binomial distribution. This distribution arrises when we let \(p\) be a random variable that follows a Beta distribution: \[\begin{align*} P &\sim \text{Beta}(\alpha, \beta)\\ X &\sim \text{Binomial}(n, P). \end{align*}\] This distribution is not part of base R, but is available using the _bbinom functions from the extraDistr package (Wolodzko 2026).

The primary benefit is that the Beta-Binomial enables us to specify some uncertainty about the skill of Player A, as \(p\) does not have to be fixed (e.g., \(p=0.70\)). Suppose the result of the best two-out-of-three match follows a Beta-Binomial distribution where \(\alpha=7\) and \(\beta=3\).

2.3 The Why

Check the expected value and variance of each distribution.

Analytically, we have \[\begin{align*} E(X) &= np &&var(X) = np(1-p) \tag{Binomial Distribution}\\ E(X) &= n\left(\frac{\alpha}{\alpha+\beta}\right) &&var(X) = \frac{n \alpha \beta (\alpha + \beta + n)}{(\alpha+\beta)^2(\alpha + \beta + 1)} \tag{Beta-Binomial Distribution} \end{align*}\]

For each distribution, compute the mean number of legs won for player A.
Hint: You can perform the sum over the finite support set here. \[\begin{align*} E(X) = \mu_X &= \sum_{i=0}^3 i f_X(i)\\ var(X) = \sigma_X &= \sum_{i=0}^3 (i-\mu_X) f_X(i) \end{align*}\]

Solution

Binomial distribution:

Code 
awinsprob=dbinom(x=2, prob = .7, size = 3)
awinsprob1=dbinom(x = 3, prob=.7, size = 3)
alosesprob=dbinom(x=1, size=3, prob=.7)
(EV=(3*awinsprob1+2*awinsprob+alosesprob))
[1] 2.1
Code 
(var=(1-EV*alosesprob)+(2-EV*awinsprob)+(3-EV*awinsprob1))
[1] 3.9567

Beta-Binomial distribution:

Code 
awinsprob=dbbinom(x=2, alpha=7, beta = 3, size = 3)
awinsprob1=dbbinom(x = 3, alpha=7, beta=3, size = 3)

Both distributions have an EV of 2.1. The variances differ slightly: Binomial distribution’s is 3.9567, while the Beta-Binomial’s is 3.9954. The Beta-Binomial distribution has a higher variance because of the added uncertainty around the chance of player A winning a match. The EV of 2.1 means that the mean number of legs won by player A is 2.1.

3 Altham’s Multiplicative Binomial Distribution

While the Beta-Binomial enables us to model cases where winning has a clustering effect (e.g., a “hot hand”).

3.1 Probability Mass Function

The mass function for this distribution is below. While it used to be available in the MBD package for R, this package is no longer available. \[f_X(x) = \frac{1}{c} {n \choose x} p^x (1-p)^{n-x} \theta^{x(n-x)}I(x \in \{0, 1, ..., n\})\] where \[c = \sum_{i=0}^n {n \choose i} p^i (1-p)^{n-i} \theta^{i(n-i)}\]

Write a function in R called daltbinom() for computing the PMF of this distribution. Use your function to plot the distribution where \(p=0.70\) where \(\theta=0.50\), \(\theta=1\), and \(\theta=1.50\). Comment on the differences in the context of a best two-out-of-three match

Solution

Here is the code and figure for when theta = .5.

Code 
p=.7
theta=.5
n=3
x=-1:4

daltbinom = function(x, p, theta, n){
  norm.c = 0
  for (i in 0:n){
    norm.c = norm.c + choose(n, i)* p^i * (1-p)^(n-i)* theta^(i*(n-i))
  }
  1/norm.c * choose(n, x) * p^x*(1-p)^(n-x)*theta^(x*(n-x))
}

Starting with theta = .5, we can see that the distribution is heavily skewed to player A winning three times. When theta = 1, we notice that the plot is exactly the same as the plot of the binomial distribution, where the probability of player A winning twice or three times is within .1, and the probability of player A winning once or no times (losing) is much smaller than the probability of winning. Comparing this to the third graph, where theta = 1.5, we can see that it becomes more likely for player A to win two games, and player A only winning one game is more likely than player A winning three. This is because theta acts as a sort of “clumping factor”, where 1 is the binomial distribution, and when theta is lower than 1, the data is more spread out, while when theta is higher than 1, the data is more clumped towards the middle values, which are one and two wins.

Write a function in R called daltbbinom() for computing the PMF of this distribution. Use your function to plot the distribution where \(\alpha=7\) and \(\beta=3\) where \(\theta=0.50\), \(\theta=1\), and \(\theta=1.50\). Comment on the differences in the context of a best two-out-of-three match.

Hint: You can use the integrate() in R to compute the integral. Note that this requires the function to be vectorized – it should be unless you have added any block conditionals.

Solution Here is the code defining my function. I reused my daltbinom function to make it easier to type out.

Code 
daltbbinom= function(x, theta, n, alpha, beta)
    {
  integrate(f = function(p) {daltbinom(x, p, theta, n)*
    (gamma(alpha+beta)/(gamma(alpha)*
    gamma(beta))*p^(alpha-1)*(1-p)^(beta-1))}, 
  lower = 0, 
  upper = 1
  )$value
}

Comment on the differences in the context of a best two-out-of-three match.

The binomial distribution and the negative binomial distribution have the same probabilites for all situations in the context of a best two-out-of-three match. The beta binomial distribution, however, incorporates variance into the skill and thus the win probability of player A. This effect can be seen by it being less likely for player A to win 2-0, but more likely for player A to win 2-1. The Althman binomial distribution adds a “hot-hand” effect, where winning a leg boosts the probability that player A wins more. It also incorporates theta, which functions as a clumping value. For theta = .5, correspondingly, the probability that player A wins 2-0 is approximately triple the next likelist outcome. As theta increases to 1.5, the data becomes more “clumped” away from the extreme outcomes, and the probability of A winning or losing 2-0 decreases, while the probabilities of A winning 2-1 or losing 1-2 increase substantially. The Althman beta binomial distribution combines the effects of Althman’s and the beta distribution. Variance in player A’s skill and a clumping factor are both added, and the result can be seen by A winning 2-0 being more likely in this distribution than in the normal Althman’s distribution.
