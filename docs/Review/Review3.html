Review for Unit Three

1 Introduction

In this review, we discuss probability. We use probability as a way to get an idea of what may be going on in a larger population based on a sample from that population. A random variable is a characteristic or measurement that we have for each observation. If it is quantitiative, it assumes we have numerical values. Within quantitative variables, we have continuous, which means the variable is double measured (infinite amount), and discrete, which means the variable is countable (limited amount).

The support of a variable tells us what values the variable can take on, similar to the domain from mathematics. The distribution of a variable tells us how often it takes these values.

For measuring probabilites that a variable takes on a specific value, we have two tools: the probability mass function (PMF), and the probability density function (PDF).

The PMF is used on discrete random variables. It is a function that returns the probability of observing x (a specific value). In graphical form, this is a population-level bar plot.

Here is an example PMF plot of the Poisson distribution. As can be seen, each value of x has a probability that corresponds to the height of its bar.

Code 
library(tidyverse)
lambda <- 2
 ggdat <- tibble(x = -1:10) |>
 mutate(PMF = dpois(x, lambda = lambda),
 CDF = ppois(x, lambda = lambda))

PMF.plot <- ggplot() +
 geom_linerange(data = ggdat,
 aes(x = x, ymin = 0, ymax = PMF)) +
 geom_hline(yintercept = 0) +
 scale_x_continuous("x", breaks=0:10) +
 ylab("Probability") +
 theme_bw()

PMF.plot

The PDF is used on continuous random variables. It is a function of the random variable where the probability is represented by the area under it. In graphical form, this is a population-level histogram or density plot.

This is similar to the data distribution: for categorical/discrete data, we use a relative frequency table to get the proportion of observations for each observable value, and then use a bar plot to visualize the proportions. For continuous data, we use the five number summary to describe it, and then use a histogram to visualize the distribution.

2 Discrete probability models

The cumulative distribution function (CDF) of a random variable (X), denoted by \(F_x(x)\), is defined for all x as \(F_x(x) = P(X \leq x)\).

This is an example CDF for the Poisson distribution.

The inverse CDF of a random variable X is denoted by \(F_x^{-1}(p)\). We can use the inverse CDF to calculate the \(p^{th}\) percentile: \(F^{-1}_X(p) = x\).

The PMF of a discrete random variable X is given by \(f_x(x) = P(X = x).\)

The CDF of a discrete random variable X is given by \(F_x(x) = P(X = x). = \Sigma f_X(x)\)

4 Continuous Probability Models

The CDF of a random variable, denoted by \(F_X(x)\), is defined as \(F_X(x) = P(X \leq x)\). The \(p^{th}\) percentile can be calculated using the inverse CDF, similar to the discrete case.

The PDF of a continuous random variable is defined as \(F_X(x) = P(X \leq x) = \int^{x}_{-\infty} f_x(x) dx\).

Some population-level characteristics than can be written are the mean: \(\mu_x = E(X)\), and the variance: \(var(X) = E[(X-\mu_x)^2]\)

5 Example on the Gaussian (Normal) distribution

The gaussian distribution is used to model a continuous outcome that is more likely to occur near the mean and symmetrically less likely to occur further from the mean. Outliers are relatively rare. It has two parameters: \(mu\), which is the population mean, and \(\sigma\), which is the population standard deviation.

Example: 1. \(P(X = 3076.88)\) 2. \(P(X \neq 3076.88)\) 3. \(P(X < 2500)\) 4. \(P(X \leq 2500)\) 5. \(P(X > 2500)\) 6. \(P(X \geq 2500)\) 7. \(P(2500 \leq X \leq 3076.88)\) 8. \(P(2500 < X \leq 3076.88)\) 9. \(P(2500 \leq X < 3076.88)\) 10. \(P(2500 < X < 3076.88)\)

6 Distributions

The Poisson and Gaussian distributions were previously discussed. In this section, we discuss many more different distributions. To model these distributions, we used the distribution functions (ex. binom, pois) with prefixes d, p, q, and r. The prefix d calculates the PMF, p calculates the CDF, q calculates the inverse CDF, and r generates random numbers from that distribution.

6.1 Discrete Distributions

The Bernoulli distribution is used to model whether an event of interest occurs in a given trial. It is parametrized by a parameter p that is the probability of the event of interest in any trial. It has a binary outcome, and is thus referred to as the “coin-flip” distribution. Here we use the distribution functions

Code 
dbinom(x = 1, prob = 0.25, size = 1) # P(X = 1 |p=0.25)
[1] 0.25
Code

# Essentially, P(X = x).

pbinom(q = 1, prob = 0.25, size = 1) # P(X <= 1 |p=0.25) 
[1] 1
Code 
# q is the bar for P(X <= q), prob is the prob of observing 1, size is number of trials again.


qbinom(p = 0.2, prob = 0.25, size = 1) # The 20th percentile
[1] 0
Code 
# p is the percentile you want to be found, prob and size are the same as previously.

rbinom(n = 10, prob = 0.25, size = 1) # A random sample
 [1] 0 1 1 0 0 1 0 1 1 0
Code 
# n is the number of random events, prob and size are the same as before.

The Binomial distribution is used to model the count of events of interest in a set of trials. It is essentially multiple Bernoulli distributions. It is parametrized by p: the probability of the event of interest occuring in any trial, and n: the number of trials.

Here we use the distribution functions:

[1] 0.1877117

pbinom(q = 1, prob = 0.25, size = 10) # P(X <= 1 |n=10, p=0.25)
[1] 0.2440252
Code 
# q is the bar in P(X <= q), prob and size are the same as before. This gives the height at the point on the CDF where x = 1. 

qbinom(p = 0.20, prob = 0.25, size = 10) # The 20th percentile
[1] 1
Code 
# p is the percentile desired, prob and size are the same.

Code 
# n is the number of times the entire experiment is completed, prob is the same, and size is the number of independent trials in a single experiment
